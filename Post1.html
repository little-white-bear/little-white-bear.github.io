pelicanとgithubを使ってブログ公開してみた。
備忘のため、ブログ更新方法をメモします。

contentフォルダ配下を更新した後、blogフォルダで以下コマンドを実行する。 

$ make publish  
$ ghp-import output  
$ git push https://github.com/kkumamon/kkumamon.github.io.git gh-pages:master

push前に編集結果を確認する場合は、

$ make html
$ make serve

を実行した後、「http://localhost:8000/」にブラウザにアクセスする。
