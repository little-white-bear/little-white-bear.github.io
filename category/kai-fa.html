Jenkinsインストール／構築①

Le Par B  | Catégorie : 開発

Tags : 投稿 /

 
$ brew install jenkins
$ brew services start jenkins

ブラウザでhttp://localhost:8080/にアクセスすると、
Unlock Jenkinsのページが表示される

~/.jenkins/secrets/initialAdminPassword
の内容をパスワード欄にコピーペーストしてcontinueを押下

推奨プラグインをインストール

ユーザ作成画面が表示される
ユーザ名：自分の名前
PW：MacログインPWと同一
メール：yahooのやつ
